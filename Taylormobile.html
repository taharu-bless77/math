関数の振る舞いに関する説明

概要

関数 \( f(x) \) が少なくとも \( n \) 回微分可能であり、次の条件が成り立つとします：

  • \( f^{(k)}(a) = 0 \)（\( k = 0, 1, \ldots, n-1 \)）
  • \( f^{(n)}(a) \neq 0 \)

このとき、\( n \) の偶奇によって \( f(x) \) の \( x = a \) 近傍での振る舞いが異なります。以下、\( a - ε < x < a + ε\)とする。

\( n \) が偶数の場合

主要項は以下のように表されます：

\[ f(x) = f(a) + \frac{f^{(n)}(a)}{n!}(x-a)^n + O((x-a)^{n}) \]

- \( f^{(n)}(a) > 0 \): \( f(x) \) は \( x = a \) を中心にして極小値を持ちます。
- \( f^{(n)}(a) < 0 \): \( f(x) \) は \( x = a \) を中心にして極大値を持ちます。

結論として、\( n \) が偶数の場合、\( f(a ± ε) - f(a)\) の正負は 微分係数 \( f^{(n)}(a)\)にのみ依存するため、\( x = a \) で明確な極値を持ちます。

\( n \) が奇数の場合

主要項は以下のように表されます：

\[ f(x) = f(a) + \frac{f^{(n)}(a)}{n!}(x-a)^n + O((x-a)^{n}) \]

\( f^{(n)}(a) > 0 \) の場合

主要項は次のように正となります：

\[ f(x) - f(a) = \frac{f^{(n)}(a)}{n!}(x-a)^n + O((x-a)^{n}) \]

・ \( x < a \) の時、\( (x - a)^{n} < 0 \) より、 \( f(x) < f(a) \)
・ \( x > a \) の時、\( (x - a)^{n} > 0 \) より、 \( f(x) > f(a) \)
・ ただし、\[O((x-a)^{n})\]は、\(x = a\) の近傍で、０に近い値をとるので、０とみなしてもよいと認める。

結論: \( x = a \) は極小のように見えますが、通過して上昇していく形になります。

\( f^{(n)}(a) < 0 \) の場合

主要項は次のように負となります：

\[ f(x) - f(a) = \frac{f^{(n)}(a)}{n!}(x-a)^n + O((x-a)^{n}) \]

・ \( x < a \) の時、\( (x - a)^{n} < 0 \) より、 \( f(x) >f(a) \)
・ \( x > a \) の時、\( (x - a)^{n} > 0 \) より、 \( f(x) < f(a) \)

結論: \( x = a \) は極大に見えますが、通過して下降していく形になります。

結論として、\( n \) が奇数の場合、\( f(x) \) は \( x = a \) で極値を持たず、単調に変化します。
